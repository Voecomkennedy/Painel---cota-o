Dicas de Backup
  • Seus dados ficam salvos automaticamente no navegador (LocalStorage)
  • Se limpar o cache ou trocar de navegador/computador, os dados são perdidos
  • Exporte o backup e salve no Google Drive para ter acesso de qualquer lugar
  • Faça backup pelo menos uma vez por semana ou após cotações importantes
  • Para usar em outro computador: exporte aqui, leve o arquivo e importe lá
